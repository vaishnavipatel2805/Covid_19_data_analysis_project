# 🦠 COVID-19 Data Analysis & Dashboard Project

## 📌 Project Overview

This project focuses on analyzing the global impact of COVID-19 using Python and visualizing key insights through an interactive Power BI dashboard. The analysis highlights trends in confirmed cases, recoveries, deaths, and vaccination progress, with a special focus on India.

The goal of this project is to derive meaningful insights from real-world data and present them in a clear, visual, and decision-friendly format.

---

## 🎯 Objectives

* Analyze global COVID-19 trends over time
* Compare confirmed, recovered, and death cases
* Study the impact of COVID-19 in India
* Visualize vaccination progress
* Build an interactive dashboard for better understanding

---

## 🛠️ Tools & Technologies Used

### 🔹 Python (Data Analysis)

* Pandas – Data manipulation and cleaning
* NumPy – Numerical operations
* Matplotlib & Seaborn – Data visualization
* Jupyter Notebook – Development environment

### 🔹 Power BI (Dashboard)

* Data modeling
* Interactive visualizations
* KPI indicators
* Filters and slicers

---

## 🔍 Data Analysis Process

### 1. Data Collection

* Collected COVID-19 datasets from reliable sources (e.g., WHO, Kaggle, government portals)

### 2. Data Cleaning

* Handled missing values
* Converted data types
* Removed duplicates

### 3. Exploratory Data Analysis (EDA)

* Time-series analysis of cases
* Country-wise comparisons
* Correlation analysis

### 4. Visualization

* Line plots for trends
* Bar charts for comparisons
* Heatmaps for correlations

* 🇮🇳 India-specific analysis

## 🚀 How to Run the Project

### Python Analysis

1. Clone the repository
2. Install required libraries:

   ```
   pip install pandas numpy matplotlib seaborn
   ```
3. Open Jupyter Notebook:

   ```
   jupyter notebook
   ```
4. Run `covid_analysis.ipynb`
